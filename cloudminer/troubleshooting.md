# Troubleshooting Guide

## Issue 1: Unable to connect

### Causes
- Incorrect POD URL  
- Network issues  
- Invalid credentials  

### Solution
- Verify URL  
- Check internet connection  
- Re-enter credentials  

---

## Issue 2: Query not executing

### Causes
- Syntax error  
- Invalid connection  

### Solution
- Check SQL syntax  
- Reconnect session  

---

## Issue 3: Application not opening

### Causes
- Installation issue  

### Solution
- Reinstall application
