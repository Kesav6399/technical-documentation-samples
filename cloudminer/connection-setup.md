# CloudMiner Connection Setup Guide

## Objective
Help users create a connection to Oracle Cloud instance.

---

## Steps

1. Open CloudMiner application  
2. Navigate to:
   - Tools → Manage Connections  
3. Click:
   - "New Connection"  

---

## Enter Connection Details

- Connection Name: User-defined  
- Connection Type: Direct  
- POD URL: Oracle Cloud instance URL  

---

## Final Steps

- Click Apply  
- Save connection  

---

## Result
Connection is successfully created and ready for usage.
